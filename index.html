處真實據點，旨在協助張伯伯在熟悉的國光社區中，克服公寓上下樓梯的挑戰，維持健康且有尊嚴的老後生活。 \end{tcolorbox} \section{一、張吉雄伯伯的生活圈資源清單} \subsection{1. 日照與居家服務 (針對張伯伯環境設計)} \begin{itemize} \item \textbf{立心社區日間照顧中心}：台北市萬華區和平西路三段122號。 \item \textbf{龍山老人服務中心}：台北市萬華區梧州街36號（適合張伯伯參與長青學苑）。 \item \textbf{愛照護萬華日照中心}：台北市萬華區長沙街二段151號。 \item \textbf{中化銀髮居家服務}：提供到府協助張伯伯家務整理與環境清潔。 \item \textbf{聖若瑟居家長照機構}：萬華區東園街140巷7號（距離張伯伯住處近）。 \item \textbf{福圓日間照顧中心}：台北市萬華區萬大路423巷。 \item \textbf{萬華區ABC長照服務站}：提供個案管理，連結爬梯機補助。 \item \textbf{慈濟居家長照機構}：台北市萬華區莒光路222號5樓。 \item \textbf{紅十字會博愛居家長照}：台北市萬華區康定路62號11樓。 \item \textbf{老吾老居家服務部}：協助張伯伯代購國光市場食材。 \end{itemize} \subsection{2. 醫療與健檢資源 (就近照護免奔波)} \begin{itemize} \item \textbf{萬華醫院}：西藏路222號（張伯伯復健與慢病管理首選）。 \item \textbf{西園醫院}：台北市萬華區西園路二段270號。 \item \textbf{萬華區健康中心}：提供張伯伯年度免費成人健檢。 \item \textbf{東園診所}：東園街172號（在地老牌家醫，回診方便）。 \item \textbf{祥恩藥局}：萬大路352號（藥師親切，可領取慢分箋）。 \item \textbf{萬大安康藥局}：萬大路442號（提供輔具與血壓測量服務）。 \item \textbf{仁濟醫院}：台北市萬華區廣州街243號。 \item \textbf{德昌藥局}：德昌街225號（提供藥物整理服務，預防吃錯藥）。 \item \textbf{聯合醫院和平院區}：設有銀髮友善門診，適合重大檢查。 \item \textbf{台北市南區輔具中心}：協助評估張伯伯家中無障礙環境改善。 \end{itemize} \subsection{3. 社交、學習與共餐 (豐富社交生活)} 包含 \textbf{萬大社區發展協會}（共餐）、\textbf{青年公園}（每日晨運）、\textbf{忠勤里食堂}及\textbf{東園里辦公處據點}，讓張伯伯走出家門與鄰里互動。 \subsection{4. 採買與休閒 (重溫回憶之旅)} 包含 \textbf{國光市場}、\textbf{雙和市場} 採買新鮮食材；體力許可時，可搭乘 \textbf{重慶幹線} 前往天母或士林官邸散心。 \newpage \section{二、給張吉雄伯伯與家屬的照顧問答} \begin{enumerate} \item \textbf{伯伯住 3 樓沒電梯怎麼出門？} 可申請長照補助的「爬梯機服務」或特約復康巴士。 \item \textbf{張伯伯想自己煮飯但體力有限？} 居服員可到府協助「備餐」，並由營養師指導慢病飲食。 \item \textbf{家中佛龕需要清潔怎麼辦？} 居服員可協助日常居家環境清潔，包含長輩重視的祭祀空間周邊。 \item \textbf{601公車現在變什麼了？} 現在更名為「重慶幹線」，班次更密集，適合伯伯重遊天母。 \item \textbf{什麼是喘息服務？} 當家人需要休假時，由政府派員接手照顧張伯伯。 \item \textbf{輔具補助有多少？} 每三年有 4 萬元額度，可用於購置助行器或改裝浴室扶手。 \item \textbf{長照申請電話？} 請撥打 1966，說明是萬華區張吉雄伯伯的需求。 \item \textbf{伯伯跌倒怎麼辦？} 建議申請「緊急救援系統」，佩戴隨身通報按鈕。 \item \textbf{共餐服務在哪裡？} 萬大社區發展協會與鄰近里辦公處皆有提供，歡迎張伯伯參加。 \item \textbf{藥太多怕吃錯？} 可請藥師到府進行「藥事照護」，整合所有處方藥物。 \end{enumerate} \vspace{2cm} \begin{center} \textbf{長照諮詢專線：1966} \\ \textit{萬華區在地安老平台 ‧ 關心張吉雄伯伯的每一天} \end{center} \end{document}
